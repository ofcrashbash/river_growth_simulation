Detailed Description

River::Border has two purposes - handle boundary of region which is futher used by River::Solver for setting boundry conditions and another one - it provides information about coordinates of sources and its dirrection of growth(perpendicular to boundary). Boundary can be initialized by Border::MakeRectangular() to rectangular region. Or it can read *.msh file by calling Border::ReadFromFile() function.

Border is as wrapper of tethex::Mesh object.
