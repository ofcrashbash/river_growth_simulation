Detailed Description

This class implements the most part of functionality of all elements of mesh: triangles, tetrahedra, quadrangles, hexahedra, etc.

All these elements are declared as pointers to base (this) class. It's not an abstract class, because it has no pure virtual functions, but you can't create objects of this class, because its constructor is protected.
