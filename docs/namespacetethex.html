double tethex::cell_measure_2D ( const std::vector< Point > &  vertices,
const std::vector< int > &  indices

Since this project was originally designed to connect Gmsh's meshes with deal.II solvers, the main requirement for new meshes was to be properly read by deal.II.

Therefore before creating quadrangle we check, that it's correctly numerated. And we check it like deal.II authors do - using cell_measure. This procedure is taken from deal.II sources.
