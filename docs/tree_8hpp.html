Detailed Description

Holds all functionality that you need to work with tree of river, its separate branches and generation of final boundary geometry.

This file holds several classes like BranchNew, Tree, GeometryNew. BranchNew class represents single branch without any biffuracation points. These branches are combined into Tree by means of Tree class. And finnaly Tree class and Border class are generated into geometry that is used in simmulation by means of GeometryNew class.
