Detailed Description

Incidence matrix describes the relations (connections) between mesh nodes.

It's used for edge numeration and for fast finding of edge number knowing 2 vertices, that define the edge. Because this matrix is symmetric we consider its lower triangle only.

Find a serial number of the non zero element in the matrix.

This number usually coincides with the number of edge. Since we keep only lower triangle of incidence matrix, row_number has to be bigger than col_number.

Parameters
row_number- the number of row where we seek (or one edge vertex)
col_number- the number of column where we seek (or another edge vertex)
Returns
Serial number of the non zero element in the matrix.
